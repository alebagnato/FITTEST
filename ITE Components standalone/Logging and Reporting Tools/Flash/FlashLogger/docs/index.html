FlashLogger (1.3)

Install

Download the zip from e.g. the library's site. Just unzip it; it should contain the following components:
automation.swc
AutomationLoader.swf
flashloggerLib.swc
FlexDelegates.swf
The .swc are needed to build your own MyLogger. The .swf need later to be deployed along with your application. The usage and deployment are explained in the documentation below.

How to Build

If you want to build the components yourself, adjust the Makefile in the root directory of the FlashLogger's source base, then just run make and make copy. This will produce the following components in ./bin:
automation.swc
AutomationLoader.swf
flashloggerLib.swc
FlexDelegates.swf
You need Flex SDK >= 4.0.

Example

One example of customizing and deploying the logger is provided in ./example. To build it, go to the folder, and run make. You then need to deploy the directory to be served by a web server. Two pages are provided:
  1. flexstore_O.html, which will load the original Flash application, without the logger.
  2. flexstore_A.html, which will load the original Flash application, with the logger attached to it.
