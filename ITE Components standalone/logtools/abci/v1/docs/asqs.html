ActionScript Quick Start

Class: an example

Here is an example of how to define a class A as part of the package spongebob.org:
// file A.as 
package spongebob.org {

   public class A {

      public var x : int ;

      public function A() { x=3 ; }

      public function foo() : int { return x*10 ; }
  }
}

Program

The common way to run an AS program is by using an Adobe's Flash player (either a stand alone version, or plugged in a web browser). Flash however does not have a console-mode. So, we cannot have a console-based Hello World example either. The simplest program will have to be either: (1) a GUI-based program, or (2) a 'sprite'. Here is a sprite example:
// file HelloWorld.as 
package {
 import flash.display.*;
 import flash.text.*;
 public class HelloWorld extends Sprite {
   private var greeting:TextField = new TextField();
   
   public function HelloWorld() {	  
     greeting.text = "Hello World!";
     greeting.x = 100;
     greeting.y = 100;
     addChild(greeting);
   }
 }
}
A GUI-based program ala Java's Swing is usually written in MXML, rather than directlyin AS. MXML is an XML-like language ontop AS. Here is an example:
// File HelloWorld.mxml 
< ?xml version="1.0" encoding="utf-8"? >
< mx:Application xmlns:mx="http://www.adobe.com/2006/mxml" xmlns="*" 
    layout="vertical"
    creationComplete = "initApp()"  >
    
    < mx:Script >
        < ![CDATA[
            
         import SomePackage.Greeter ;

         private var g:Greeter = new Greeter();
            
         public function initApp():void { mainTxt.text = g.sayHello(); }

        ]] >
    < /mx:Script >

    < mx:TextArea id= "mainTxt" width="400" / >        
    
< /mx:Application >

Compiling to an swc and swf

An swc file is like a jar file containing compiled classes. In particular, an swc is not an executable. To compile one or more source files containing AS classes and then packaging them into an swc file you can use Flex's compc compiler: 
compc  -source-path blabla/src 
       -output blabla/bin/mylib.swc 
       -include-classes classA classB classC ...
To compile a program use Flex's mxmlc compiler: 
mxmlc -library-path+= blabla/libs 
      -static-link-runtime-shared-libraries=true 
      -source-path blabla/src
      -output blabla/bin/helloworld.swf 
      -- blabla/src/helloworld.as

Running a program

After compiling a program into an swf file, you can load it into a Flash player to run it. You may need to change the setup of the player to trust the swf file (or the directory it is is).

Tracing

Use the function trace("hi") to produce tracing information. If you run the program with the Flash player, it will write the traced info into its log file. You need to first setup the player to enable logging.
